# CAD

3D-printed parts for the heated insert press. All STL.

| Part | Role |
|---|---|
| `arm.stl` | Arm that clamps the soldering iron |
| `clamp-mount.stl` | Mounting block for the soldering-iron clamp |
| `clamp-cover.stl` | Cover for the clamp |
| `roller-plate.stl` | Carriage plate riding the 2020 extrusion on ball-bearing wheels |
| `roller-handle.stl` | Handle on the carriage for manual press actuation |
| `pulley-wheel.stl` | Pulley for the counterweight cable |
| `idler-mount.stl` | Mount for the counterweight idler |
| `idler-holder.stl` | Holder bracket for the idler bearing |
| `idler-coupler.stl` | Coupler joining idler components |
| `idler-cover.stl` | Cover for the idler assembly |
| `weight-holder.stl` | Counterweight container |
| `mount.stl` | Base mount securing the platform bracket to the 2020 extrusion |

## Notes

- Designed around standard 2020 aluminum extrusion.
- Carriage uses ball-bearing wheels riding the extrusion slots.
- A T-nut acts as the lower travel end-stop so the iron tip never hits the platform.
- Compatible with any soldering iron + heat-set installation tip clamped into `arm.stl`.
