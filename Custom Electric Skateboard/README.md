# Custom Electric Skateboard

High-performance electric skateboard built on a repurposed deck.

![Completed Electric Skateboard](https://github.com/user-attachments/assets/b3e8b0a6-c25f-4a0b-80eb-266fd92d0fd4)

## Specifications

| Spec | Value |
|---|---|
| Range | ~30 mi per charge |
| Top speed | 35 mph |
| Motor | 6380 BLDC, 190 KV |
| ESC | FOCBOX Unity |
| Battery | Custom 12s3p Samsung 40T Li-ion |
| Drivetrain | Belt drive |

## Battery pack

- 12s3p configuration (36 cells, ~44 V nominal)
- Samsung 40T 21700 cells
- Hand spot-welded
- Integrated BMS for charge balancing and protection

## Drivetrain

- 6380 BLDC, 190 KV
- FOCBOX Unity ESC (dual-motor capable, used single here)
- Belt-driven rear wheel for torque + serviceability

![Skateboard Drivetrain](https://github.com/user-attachments/assets/ece9ce4f-5887-48d1-8b6b-387f360bf0d2)

## Build sequence

1. Specced electrical system + selected components for 12s pack and 6380 motor
2. Hand spot-welded the 12s3p pack
3. Integrated BMS
4. Mounted motor + ESC to the deck
5. Installed belt drive
6. Wired, configured FOCBOX Unity, range-tested
