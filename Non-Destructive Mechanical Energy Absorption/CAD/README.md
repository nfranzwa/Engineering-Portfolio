# CAD

STL files for the rotational snap-through node, full lattice assembly, mounts, and machining soft-jaws.

| File | Role |
|---|---|
| `bot_assembly.stl` | Single bottom-node disc with the circular magnet array |
| `full_assembly.stl` | Full chiral-lattice test article (multiple stacked nodes) |
| `mounts.stl` | MTS tester attachment mounts |
| `softjaw_assembly.stl` | Custom mill soft-jaws used for the two-WCS prototyping setup (see `../Machining/Prototyping/`) |
